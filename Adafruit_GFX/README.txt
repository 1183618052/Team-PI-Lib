This is the core graphics library for all Adafruit displays, providing a common set of graphics primitives (points, lines, circles, etc.).  It needs to be paired with a hardware-specific library for each display device we carry (to handle the lower-level functions).

Written by Limor Fried/Ladyada for Adafruit Industries.
Modified by Brian Chen for performance on the Teensy 3.x
